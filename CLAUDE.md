# 超弾幕シューティングゲーム

Phaserで作成したブラウザ対応の縦スクロール弾幕シューティングゲーム

## ゲーム概要
- 画面左側65%がゲーム画面、右側35%がスコア・ライフ表示
- 東方紅魔郷風の弾幕シューティング
- 3種類のプレイヤーキャラクター選択可能

## プレイヤー種類
1. **プレイヤー1**: 青とオレンジの宇宙船、通常弾
2. **プレイヤー2**: 黄色の宇宙船、1秒後に爆発する特殊弾
3. **プレイヤー3**: 緑の宇宙船、ライフに応じた扇状弾（スリーウェイ）

## 操作方法

### キーボード操作
- **プレイヤーセレクト**: ← → キーで選択、↑↓で選択モード切替、Enterで決定
- **ゲーム中**: 矢印キーで移動、スペースキーで弾発射、Gキーでグレネード投擲、F6キーでプレイヤーセレクトに戻る

### ゲームパッド操作
- **プレイヤーセレクト**: 十字キー/スティック左右で選択、Aボタンで決定、Bボタンで選択モード切替
- **ゲーム中**:
  - アナログスティック（左）または十字キーで移動
  - Aボタンで弾発射
  - Bボタンでボス即出現（ボス戦時以外）、グレネード投擲（ボス戦時）
  - Xボタンでグレネード投擲
  - スタートボタンでプレイヤーセレクトに戻る

## 敵の特徴
- 通常敵（赤）: プレイヤーを狙った弾を発射
- 大型敵（暗い赤）: 16方向に弾幕を展開
- 強化敵（青、ボス撃破後出現）: 3発同時発射

## ボス
- **通常ボス**（1体目、3体目以降）: 紫色の大型敵、難易度に応じて弾幕パターンが変化
- **双子ボス**（2体目）: 勾玉の形をした2体のボス、片方が倒されると残りが怒りモードに

## ゲームシステム
- ライフ表示（ハート）
- スコアとハイスコア
- パワーアップアイテム（速度UP、HP回復）
- りんご（50秒ごとに出現、ライフ+3）
- グレネード（画面上の敵を一掃）
- ゲームオーバー画面

## 実行方法

### 開発サーバーで実行（推奨）
```bash
# 依存関係をインストール（初回のみ）
npm install

# 開発サーバーを起動
npm run dev
```
ブラウザで http://localhost:5173 を開く

### ビルド
```bash
npm run build
```
`dist/` フォルダに静的ファイルが生成される

## 技術スタック
- Phaser 3（ゲームフレームワーク）
- TypeScript
- Vite（ビルドツール）

## ファイル構成
```
src/
  main.ts                 - エントリーポイント
  constants.ts            - 定数定義
  scenes/
    PlayerSelectScene.ts  - プレイヤー選択画面
    GameScene.ts          - メインゲーム画面
    GameOverScene.ts      - ゲームオーバー画面
public/
  assets/
    audio/                - BGM、効果音
    images/               - 画像、スプライト
index.html                - HTMLテンプレート
package.json              - npm設定
tsconfig.json             - TypeScript設定
vite.config.ts            - Vite設定
```
